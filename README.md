# 微软大战代码 (VS Code Wars) 🎮

基于 "微软大战代码" 这个有趣梗开发的射击游戏！驾驶 VS Code 飞船，用代码符号武器击退入侵的 Bug 怪物！

## 关于这个梗

"微软大战代码" 实际上是对 **Microsoft Visual Studio Code** (VS Code) 的一个趣味误读。这个项目就是基于这个有趣的误解，把 VS Code 变成了一架战斗飞船，对抗各种 Bug 怪物！

## 游戏特色

- 🛡️ **VS Code 主题飞船**：驾驶标志性的蓝绿色飞船
- 🔫 **代码符号武器**：使用 `{`, `}`, `;`, `(`, `)`, `[`, `]` 等代码符号射击
- 🐛 **多种 Bug 敌人**：
  - 🐛 普通 Bug：速度适中，10分
  - 🐞 瓢虫 Bug：血量较厚，20分
  - 🕷️ 蜘蛛 Bug：速度快，15分
  - 🦟 蚊子 Bug：超级快速，25分
- 💥 **爆炸特效**：像素风格的爆炸粒子效果
- 📊 **分数和生命值系统**：追踪你的战绩
- 📦 **Package 系统**：收集不同的 package 获得特殊效果！
  - ⚡ **Vite**：召唤 2 个子编辑器跟随，增强火力！
  - ✅ **ESLint**：瞬间清屏，消灭所有敌人！
  - 🔄 **Git**：回复一条命！
- 🎭 **敌人特效**：
  - `null` 前期隐身后期显形！
  - `NullPointerException` 也会来捣乱！
- 💀 **空指针危机**：只有 `NullPointerException` 逃跑时才会让你扣血，一定要拦截住！

## 如何玩

1. 克隆这个仓库
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 在浏览器中打开游戏
5. 开始战斗！

### 操作说明

- **移动**：使用 `WASD` 键或方向键
- **射击**：使用空格键
- **目标**：消灭尽可能多的 Bug，不要让它们撞到你！

## 技术栈

- ⚡ **Vite**：快速的开发构建工具
- 🎨 **HTML5 Canvas**：高性能游戏渲染
- 📦 **Vanilla JavaScript**：原生 JavaScript，无需额外框架
- 🎨 **CSS**：精美动画和样式

## 项目结构

```
├── src/
│   ├── main.js      # 游戏入口，DOM 操作
│   ├── game.js      # 游戏状态和常量管理
│   ├── logic.js     # 游戏逻辑和碰撞检测
│   ├── renderer.js  # 渲染相关函数
│   └── style.css    # 游戏样式
├── index.html       # 主 HTML 文件
├── vite.config.js   # Vite 配置
└── package.json     # 项目信息和依赖
```

## 构建项目

```bash
npm run build
```

构建产物将输出到 `dist` 目录。
