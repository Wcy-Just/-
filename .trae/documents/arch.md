
# 微软大战代码 - 技术架构文档

## 1. 架构设计
```mermaid
graph TD
    A["HTML5 Canvas 渲染"] --&gt; B["游戏引擎"]
    B --&gt; C["输入处理"]
    B --&gt; D["游戏逻辑"]
    D --&gt; E["玩家系统"]
    D --&gt; F["敌人系统"]
    D --&gt; G["碰撞检测"]
    D --&gt; H["分数系统"]
    B --&gt; I["UI 层"]
    I --&gt; J["开始/结束界面"]
    I --&gt; K["HUD"]
```

## 2. 技术描述
- **前端框架**：原生 HTML5 + CSS3 + JavaScript
- **渲染引擎**：HTML5 Canvas 2D
- **构建工具**：无（纯静态文件）
- **后端**：无
- **数据库**：无（本地存储分数可选）

## 3. 文件结构
```
/
├── index.html        # 主页面
├── style.css         # 样式文件
├── game.js           # 游戏核心逻辑
├── player.js         # 玩家系统
├── enemy.js          # 敌人系统
├── bullet.js         # 子弹系统
└── utils.js          # 工具函数
```

## 4. 核心类/模块

### 4.1 游戏引擎 (Game)
- 管理游戏状态（开始、运行、结束）
- 主游戏循环
- 渲染调度
- 碰撞检测系统

### 4.2 玩家 (Player)
- 位置和移动
- 生命值管理
- 射击功能
- 绘制方法

### 4.3 敌人 (Enemy)
- 多种敌人类型
- 移动模式（直线、Z型、追踪）
- 生命值和伤害计算
- 生成系统

### 4.4 子弹 (Bullet)
- 多种武器类型
- 速度和伤害
- 绘制方法

### 4.5 工具函数
- 随机数生成
- 碰撞检测
- 动画辅助函数
